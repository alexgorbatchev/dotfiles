# installer-zsh-plugin Package

## Purpose

Installer plugin for zsh plugins via git clone. See [README.md](README.md) for details.

## Key Files

- `ZshPluginInstallerPlugin.ts` - Main plugin class
- `installFromZshPlugin.ts` - Installation logic
- `schemas/` - Zod schemas for validation
- `types.ts` - TypeScript types
- `log-messages.ts` - Logging messages
