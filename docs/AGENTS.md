This is a user facing documentation.

While the code has `@dotfiles/...` imports, it's published as a single package called `@gitea/dotfiles`, so all examples here must import from single `@gitea/dotfiles` package. All types and functions are exported from the package are the same as the ones in the monorepo, so you can use the monorepo documentation as a reference.

All code examples must use `defineTool`, all other APIs for defining tools are internal.
